This Application is created based on Eclipse IDE / Java 8, 

To run application:

1. Open the project and clean and wait build
2. Run Application.java under com.robot package in the com.codetest.src folder.
3. follow the instructions
4. Result will be output to console

To do unit test:

Run TestRobot.java under the package of com.robot in the folder of com.codetest.test

Log is recorded robot.log under folder RobotCodeCallenge.
